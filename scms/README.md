# Social Content Management System

**Date:** 2026-01-05

## Value
* The system optimizes time and effort of a content creater to gain optimal revenue.
* The system optimizes time and effort of a content consumer to consume a content.
* The system optimizes quality and revenue of a content

## Goal
Create an AI native solution to have a unified system for content creater, content consumer. This system will optimize value achieved by content creater, content consumer. This system should be **entertaining** for some consumers and **useful** for some consumers.

## What is it?
The system will let a content creater work on projects. The projects can be of different types
1. Standalone
1. Small
1. Medium
1. Large

The most basic project is standalone project which can be created and published instantly. The projects will slide automatically based on an optimal short duration to keep it's consumer engaged.

The consumer of the system will receive points based on completion of simple action items. Based on the points achieved the consumer will build a credential of having depth of information about different projects.

The system will share revenue with content creater. This system will also help to generate revenue for it's consumer.

## Components
* Content
* Content Creater
* Content Consumer

### Content
This system should be broad enough to include many types of content like
* Text
* Image
* Audio
* Video

### Content Creater
Content creater is an entity which creats the content. His goal is to make a content which is entertaining and useful. He takes the ownership to get the most value out of the content created.

### Content Consumer
Content consumer is an entity which consumes the content by reading, watching or listening to it.
