# Literary Management System

**Date:** 2025-12-27

## Value
The system optimizes time and effort of a writer, reader to gain optimal value. This system optimizes value of a book seller.

## Goal
Create an AI native solution to have a unified system for writer, reader and book seller. This system will optimize value achieved by writer, reader and book seller.

## Components
* Writer
* Reader
* Seller
* Learning Management System (LMS)

### Writer
A writer is simply an entity who writes in textual format in different languages. It may use drawing, images, audio and video in it's writing.

#### What the System should do to improve writer's ability?
The system should provide below features to improve writer's ability to write better.
1. Online editor that is available over the internet
1. Offline editor
1. AI functionality on the editor
1. Change Management
1. Mobile App Editor
1. Paper text to Digital text converter

#### Online Editor
The online editor should have below functionality
1. It should provide the ability to write in different global and local languages
1. It should provide ability to format the text
1. It should provide shortcut to formatting action
1. It should provide UI buttons to perform formatting


### Reader
A reader is simply who reads, listen, watches content created by a writer.


#### What the System should do to improve reader's ability?
1. It should provide an automatic indication of what is already read
1. Filtering of sections
1. Narration of textual content
