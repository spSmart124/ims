# Social Content Management System

**Date:** 2026-01-05

## Value
The system optimizes time and effort of a content creater, content consumer to gain optimal value. This system optimizes value of a content creater.

## Goal
Create an AI native solution to have a unified system for content creater, content consumer. This system will optimize value achieved by content creater, content consumer.

## What is it?
The system will let a content creater work on projects. The projects can be of different types
1. Standalone
1. Small
1. Medium
1. Large

The most basic project is standalone project which can be created and published instantly. The projects will slide automatically based on an optimal short duration to keep it's consumer engaged.

The consumer of the system will receive points based on completion of simple action items. Based on the points achieved the consumer will build a credential of having depth of information about different projects.

The system will share revenue with content creater.

## Components
* Content Creater
* Content Consumer

### Writer
A writer is simply an entity who writes in textual format in different languages. It may use drawing, images, audio and video in it's writing.

#### What the System should do to improve writer's ability?
The system should provide below features to improve writer's ability to write better.
1. Online editor that is available over the internet
1. Offline editor
1. AI functionality on the editor
1. Change Management
1. Mobile App Editor
1. Paper text to Digital text converter

#### Online Editor
The online editor should have below functionality
1. It should provide the ability to write in different global and local languages
1. It should provide ability to format the text
1. It should provide shortcut to formatting action
1. It should provide UI buttons to perform formatting


### Reader
A reader is simply who reads, listen, watches content created by a writer.


#### What the System should do to improve reader's ability?
1. It should provide an automatic indication of what is already read
1. Filtering of sections
1. Narration of textual content
